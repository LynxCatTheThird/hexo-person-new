## 附录

### Markdown-it 拓展语法

#### 注释

```
*[被注释文本]: 注释内容

这是 被注释文本
```

#### 上标与下标

```
上标^示例^

下标~示例~
```

#### 脚注

```
脚注示例[^1]

[^1]: 脚注是指附在文章页面的最底端的，对某些东西加以说明的注文。
```

#### 容器

```
::: tips
**提示**
这是一个提示
:::
```
